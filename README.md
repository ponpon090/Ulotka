# 🛡️ Ulotka o Szczepieniach

Pięknie zaprojektowana ulotka edukacyjna o szczepieniach w języku polskim, zawierająca kompleksowe informacje na temat ochrony zdrowia poprzez szczepienia.

## 📋 Zawartość

Ulotka zawiera następujące sekcje:

- **JAK DZIAŁAJĄ SZCZEPIENIA?** - Wyjaśnienie mechanizmu działania szczepionek krok po kroku
- **DLACZEGO WARTO SIĘ SZCZEPIĆ?** - Korzyści i znaczenie szczepień dla zdrowia
- **FAKTY I MITY** - Rozwiązywanie popularne mitów medycznych opartymi o fakty naukowe
- **SZCZEPIENIA OBOWIĄZKOWE W POLSCE** - Pełny harmonogram szczepień obowiązkowych
- **WEZWANIE DO DZIAŁANIA** - Motywujące wiadomości do działania

## 🎨 Cechy projektu

## 📥 Jak pobrać jako PDF

### Metoda 1: Drukowanie do PDF (rekomendowane)
1. Otwórz plik `szczepienia.html` w przeglądarce
2. Naciśnij `Ctrl+P` (Windows/Linux) lub `Cmd+P` (Mac)
3. Wybierz opcję "Zapisz jako PDF"
4. Wybierz lokalizację i kliknij "Zapisz"

### Metoda 2: Za pomocą aplikacji biurowych
1. Otwórz `szczepienia.html` w programie biurowym (np. LibreOffice)
2. Użyj opcji "Eksportuj jako PDF"
